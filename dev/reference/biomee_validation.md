# rsofun BiomeE targets validation data

Small example dataset of target observations (leaf trait data) at the
CH-LAE site to optimize model parameters with the function
[`calib_sofun`](https://geco-bern.github.io/rsofun/dev/reference/calib_sofun.md)

## Usage

``` r
biomee_validation
```

## Format

A tibble of validation data:

- sitename:

  site name

- data:

  validation data
